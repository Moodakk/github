# Dental Exam Assistant

Цей додаток допомагає стоматологам готуватися до апробації (медичного іспиту) в Чехії.
Він включає два основні інструменти:
1. **Обробка файлів (Підготовка бази знань)**: Розділяє великі PDF-файли на частини по <200MB для завантаження в NotebookLM. Включає опціональне розпізнавання тексту (OCR) чеською мовою для сканованих сторінок, видаючи результати у вигляді `.txt` файлу.
2. **Розумний Верифікатор**: Інструмент для перевірки тестових питань з використанням моделей штучного інтелекту (Gemini, Claude, ChatGPT). Використовує суворий промпт згідно з правилом "все або нічого" (all or nothing), запобігаючи галюцинаціям. Ви можете використовувати його для генерації промпту для ваших веб-версій (Pro) або підключити API ключі для перевірки прямо в додатку.

## Як запустити на вашому комп'ютері:

1. Встановіть Python (версія 3.9 або новіша).
2. Відкрийте термінал / командний рядок у цій папці.
3. Встановіть необхідні бібліотеки:
   `pip install -r requirements.txt`
4. Встановіть Tesseract OCR на ваш комп'ютер (і не забудьте додати підтримку чеської мови `ces`):
   - **Для Windows**: Завантажте інсталятор з https://github.com/UB-Mannheim/tesseract/wiki і під час встановлення виберіть Czech language data.
   - **Для Mac**: `brew install tesseract tesseract-lang`
   - **Для Linux/Ubuntu**: `sudo apt install tesseract-ocr tesseract-ocr-ces`
5. Запустіть програму:
   `streamlit run app.py`
6. Програма відкриється у вашому веб-браузері.
